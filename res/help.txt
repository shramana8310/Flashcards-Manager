====================
Flashcards Manager 시험판 도움말
====================
본 프로그램은 Flashcards Deluxe와 같은 단어장 암기 어플에서 사용할 단어장 파일을 손쉽게 제작하기 위한 프로그램입니다.
본 프로그램을 이용하기 위해서는 단어 목록 파일, 스타딕 파일을 준비해야 합니다.

1. 단어 목록 파일
암기할 단어 목록을 한 줄에 한 단어씩 기재한 텍스트 파일입니다. 파일 크기에 제한은 없습니다. 다음 예시를 참고하십시오.

[예시: fruits.txt]
apple
banana
elderberry
clementine
fig
guava
mango
watermelon

2. 스타딕(StarDict) 파일
단어의 뜻을 찾는 데에 참조할 사전 파일입니다. 본 프로그램에서 스타딕 파일을 지정할 때에는 ifo 확장자 파일을 지정하십시오.
스타딕에 대한 자세한 사항은 다음 URL을 참조하십시오.
http://stardict.sourceforge.net/

====================
단어장 파일에 대해
====================
"단어장 파일"은 결과물이 출력될 파일입니다. 적당한 경로를 지정하면 됩니다.
단어장 파일은 헤더가 'Text 1', 'Text 2'인 TSV(tab-separated values) 파일로 저장됩니다.
추가적인 편집이 필요한 경우 텍스트 편집기나 MS 엑셀 등을 활용하시면 됩니다.

====================
Flashcards Deluxe에서 단어장 파일을 사용하는 방법
====================
Flashcards Deluxe가 설치된 단말(iOS 단말, 안드로이드 단말 등)에 USB 등으로 직접 단어장 파일을 옮기거나, Dropbox의 Flashcards Deluxe 폴더에 단어장 파일을 업로드하면 Flashcards Deluxe 앱에서 결과물을 확인할 수 있습니다.

====================
문제 발생시
====================
Q. 본 프로그램으로 작업을 개시했을 때, 인코딩 관련 문제가 발생합니다.
A. 메모장과 같은 텍스트 편집기로 스타딕의 ifo 파일을 열어서 영자, 숫자가 아닌 문자열(가령 한글 문자)을 삭제하거나 적당히 수정한 뒤 재시도 해 봅니다.
해당 문자열을 삭제했음에도 여전히 문제가 발생할 경우 개발자에게 알려주시면 감사하겠습니다.

본 프로그램은 어디까지나 개인이 취미로 개발한 시험용 프로그램이며, 문제점이 발견될 경우 향후 업데이트를 통해 해결하도록 하겠으나 해결될 것이라는 보장은 없습니다.
이 점 양해 부탁드립니다.
